                            QUICK START
===============================================================================

To Deploy to Vercel:

  1. Ensure git is updated:
     cd /Users/chapo/Downloads/seo-mothra-improved
     git status  (should be clean)

  2. Set up Vercel:
     npm i -g vercel
     vercel deploy

  3. Follow Vercel prompts (choose project scope, etc.)

  4. Set environment variables in Vercel Dashboard:
     NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id
     NEXT_PUBLIC_SANITY_DATASET=production

  5. Visit your site at https://[project].vercel.app

To Run Locally:

  1. Install & build:
     npm install
     npm run build

  2. Start production server:
     npm run start
     # Runs on http://localhost:3000

  3. Or dev mode:
     npm run dev
     # Hot reload on changes
