## New Design System

### Color Palette

**Light Theme Foundation:**
- **Background**: `#faf7f2` (off-white cream)
- **Accent Sage**: `#7a9b6d` (primary, calming green)
- **Accent Rose**: `#c97b7b` (secondary, warmth)
- **Accent Gold**: `#c4a574` (tertiary, elegance)
- **Text**: `#2a2622` (deep charcoal)
- **Muted**: `#6b6560` (soft gray)

**Backgrounds with Depth:**
- Primary gradient: `#faf7f2 → #f5f1eb → #f0ebe5`
- Subtle grain texture overlay (4% opacity)
- Glassmorphism cards with backdrop blur

### Typography System

**Headlines:** `Outfit` (modern, geometric, bold)
- 72px hero (desktop)
- 48px section headers
- 24-28px card titles
- Bold weights: 700-800

**Body:** `Inter` (clean, accessible, neutral)
- 16-18px body text
- 14px secondary text
- 12px labels
- Font weights: 300-600

### Visual Elements

**Cards & Containers:**
- Soft rounded corners (16-24px border-radius)
- Subtle borders: `rgba(122, 155, 109, 0.15-0.3)`
- Background: `rgba(255, 248, 238, 0.6)` with backdrop blur
- Hover states: Lift effect, border color increase, soft shadow
- No harsh shadows—all shadows use soft blur (lg/xl)

**Buttons:**
- Rounded full (`border-radius: 9999px`)
- Primary: Sage background, white text
- Secondary: Sage border, sage text, hover: light sage background
- Smooth transitions (0.3s cubic-bezier)

**Spacing:**
- Generous whitespace (16-24px between sections)
- Grid gaps: 24-32px
- Padding: 32-48px on main sections
- Mobile: 16-20px padding
